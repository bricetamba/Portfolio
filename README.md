# Brice Tamba Prono — Portfolio Website

A premium personal brand website for a Pharmaceutical Brand Manager.

## Stack
- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **next-themes** (dark/light mode)
- **Lucide React** (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Sections
- **Hero** — Animated landing with CTAs
- **About** — Professional summary + expertise pillars
- **Experience** — Interactive expandable timeline (4 roles)
- **Impact** — Achievement cards (8 initiatives)
- **Skills** — Tabbed badge browser (5 categories)
- **Education** — MSc cards + certifications
- **Volunteering** — Football Coach + Firefighter
- **Contact** — Email, LinkedIn, Phone, Location

## Customization
1. Update contact details in `src/components/sections/Contact.tsx`
2. Add your actual `public/resume.pdf`
3. Update LinkedIn URL in Hero + Contact sections
4. Adjust phone/email placeholders

## Design Tokens
- Primary: `#0093D0` (Pfizer Blue)
- Navy: `#0A1628`
- Dark mode by default, toggleable to light
